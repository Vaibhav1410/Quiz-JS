About Us😀

The Quiz App in Web is a simple Quiz App that contains a set of curated questions and its answers and checks for the correctness of the answer given by the user. It navigates through the questions using dynamic programming.

Quiz App is an Web app that lets you take a series of tests on the HTML,CSS and Javascript language. People who use the app will find it very easy to answer the multiple-choice questions because the app is very simple to use. At the end of the quiz, a report is made that shows the score.
